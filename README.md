# Major Project: Lung Disease Detection System

This repository contains the code for a lung disease detection system using deep learning and Django.

## Features
- Deep Learning models (MobileNet, ResNet-50) for lung disease classification.
- Django-based web interface for uploading images and viewing results.
- Automated report generation.

## Setup
1. Clone the repository.
2. Install dependencies.
3. Run `python manage.py runserver`.
